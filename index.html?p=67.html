Downloads

Databases engines

Download Apache Cassandra version 1.x, not 2.x : here (see Previous and Archived Cassandra Server Releases) or Debian version (choose 12x series).

Download ElasticSearch here or Yum/Debian version.

You should use a Java 7 JVM from Oracle.

Direct pack for Linux & OSX

$ cd /opt
$ curl -s http://mydmam.org/dwl/pack.sh | sh

You need java, git, curl and tar to start it.

Else

Download Play! Framework (MD5: ea1c40c3563c57c276d898f5af456f92)

Download MyDMAM from github master zip , or

git clone https://github.com/hdsdi3g/mydmam.git

Download startup scripts. For the moment, it’s only for Linux/Debian.

Download dependencies (if you don’t want compile all), to add to MyDMAM lib/ directory.

Warning : don’t forget to get all your actual ElasticSearch jars (from setup) for add to lib directory, then remove old elasticsearch-0.xx.x (or Lucene) versions from package. Don’t let 2 versions for a jar in MyDMAM lib directory, and don’t let different versions between ElasticSearch server and MyDMAM lib directory. By the way, use the same JVM with MyDMAM and ElasticSearch.
