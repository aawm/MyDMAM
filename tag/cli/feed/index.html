It retrieve loudness (LUFS) history for any media file (audio and video). The analyst is in the backend side, and it displays results in web interface some audio stats (like LU, True Peak, DC offset, RMS average…) and an interactive graphic. The backend engine is libebur128 via ffmpeg and the graphic engine is jFreeChart.
