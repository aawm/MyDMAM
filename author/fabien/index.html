MyDMAM a 2 ans

12/10/2014Newshdsdi3g

MyDMAM a aujourd’hui 2 ans, et le code est publié depuis 1 an.

Depuis un an, il y a eu quelques nouvelles fonctionnalités, des ré-écritures de code (car tout ne peut pas être parfait du premier coup), ainsi que pas mal de correctifs, notamment par rapport aux connexions avec les bases de données.

Cet été a marqué l’ajout des UserAction, et depuis quelques semaines la ré-écriture de tout le système de gestion de tache interne afin de pouvoir débloquer des évolutions futures. La branche Master ne sera mise à jour que quand il y aura quelque chose de cohérent car je ne veut pas mélanger code à tester, code fonctionnel et code mort dans Master.

Donc, le développement est toujours actif. Le projet est toujours vivant, toujours en alpha, et avance progressivement.
